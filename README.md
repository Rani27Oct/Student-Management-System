# Student Management System

This is a simple Student Management System built using HTML, CSS, and JavaScript.

## Features
- Add student details
- Display student list
- Edit student information
- Delete student
- Data stored using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## Future Scope
- Connect with backend using Java (Spring Boot)
- Store data in database (MySQL)

 ## How to Run
1. Download or clone the repository
2. Open index.html in any browser (Chrome recommended)
3. Start adding student details
